# Sheetify

Sheetify is a professional-grade Intelligent Document Processing (IDP) application designed to streamline data entry by extracting structured tabular data from unstructured documents. 

Built with a modern stack including **React**, **TypeScript**, **Express**, and high-performance **Large Language Models**, Sheetify provides an intuitive interface for transforming complex documents into organized, exportable spreadsheets.

## Key Features

- **Multi-Document Synthesis**: Upload multiple images or pages and have them consolidated into a single structured dataset.
- **AI-Powered Insights**: Get executive summaries, key business metrics, and categorical distributions automatically.
- **Multilingual Support**: Process documents in various languages with automatic header normalization.
- **Interactive Workbench**: Edit extracted data directly in a smart grid with double-click inline editing.
- **Enterprise Exports**: Export your data to Excel (.xlsx), CSV, or JSON for seamless business integration.
- **Document History**: Maintain a searchable history of processed documents for quick reference and collation.

## Technical Stack

- **Frontend**: React 18, Vite, Tailwind CSS, Motion (Framer Motion), Recharts, Lucide Icons.
- **Backend**: Node.js, Express, Multer.
- **Intelligence**: Advanced multimodal model integration for high-performance data extraction.
- **Exporting**: ExcelJS for high-fidelity spreadsheet generation.

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set your `GEMINI_API_KEY` in the environment variables.
4. Start the development server: `npm run dev`.
5. Open your browser and start processing documents.
